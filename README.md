# Sale_dashboard
Sale Dashboard for checking KPIs for each REP. This dashboard analyze the revenue across regions, focusing on trends and forcast revenue.
## Tools used
- Power BI
- Excel
- Python
- SQL
## Key Features
- Revenue & Budget trends weekly by REPs
- Trend and forecast identification
- Interactive filters (e.g., REPs, year & month)
##  Files
- `Sale Dashboard_py.pbix` – Power BI report file containing the full sales dashboard.
- `Sale transaction current month data.xlsx` - Dataset with the current month’s sales transactions. All values are randomly generated.
- `Sale transaction historical data.xlsx` - Dataset with historical sales transactions. All values are randomly generated.
- `dim_tam.xlsx`
- `dim_tambudget.xlsx`
